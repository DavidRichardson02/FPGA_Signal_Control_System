FPGA Signal Control System

Real-Time Mapping • Sensor Fusion • Hardware Control • VGA Visualization

Overview

This project implements a fully hardware-accelerated sensing and control system on the Digilent Nexys A7 (Artix-7 FPGA). The design integrates Time-of-Flight ranging, PIR motion detection, temperature sensing (XADC + TMP), rotary encoder surveying, dual-buffered VGA graphics, and a hardware-deterministic UART telemetry pipeline.

Core Features

  • Real-time ToF range acquisition (ISL29501) with IRQ-driven measurement
  • 360° surveyor FSM with rotary encoder manual override
  • Full VGA pipeline: timing, range plot, HUD, logo buffer
  • Temperature-driven proportional fan controller
  • PIR “recent-motion” fading indicator (RGB LED + HUD widget)
  • High-speed UART telemetry (2 Mbit/s) + MATLAB live decoding
  • Double-buffered 320×240 image loader via UART

System Architecture

Telemetry & MATLAB Tools

All measurements (distance, angle, status, temperature, fan duty) are streamed in 17-byte frames. MATLAB scripts included in the repository provide real-time:

  • Distance-vs-time plots
  • Polar mapping
  • Status timelines
  • Automatic CSV logging
  • VGA image loader (RGB444 → UART → FPGA)
